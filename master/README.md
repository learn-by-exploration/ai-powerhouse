# Master — AI Powerhouse Setup

This folder is the unified entry point for using all tools in the AI Powerhouse collection inside Claude Code.

See **[USAGE.md](../USAGE.md)** for the full usage guide.

---

## What's inside

```
master/
├── CLAUDE.md              # Master instructions — loaded automatically by Claude Code
├── install.sh             # Install all tools into ~/.claude
├── uninstall.sh           # Remove everything install.sh placed in ~/.claude
├── hooks/
│   └── ruflo-hooks.json   # Version-pinned ruflo hooks (claude-flow@3.5.0)
├── skills/
│   └── agent-routing/     # Decision tree for routing tasks to the right tool
└── .claude/               # Auto-generated by install.sh (gitignored)
    ├── agents/            # 123 agents (ecc-*, superpowers-*, gsd-*, ruflo-*)
    ├── skills/            # 189 skills
    ├── commands/          # 68 commands
    ├── hooks/             # Hooks from ecc, gsd, claude-mem, ruflo
    └── rules/             # Common + optional language rules
```

---

## Quick start

```bash
# Preview
bash master/install.sh --dry-run

# Install (minimal, recommended)
bash master/install.sh

# Install without ruflo enterprise agents (saves ~40-60K context tokens)
bash master/install.sh --no-ruflo

# Full install (adds language-specific rules)
bash master/install.sh --full
```

Restart Claude Code after installing.

## Install flags

| Flag | Effect |
|------|--------|
| _(none)_ | Minimal install — core tools, no language rules |
| `--full` | + 11 language rule sets (~15-20K more tokens) |
| `--no-ruflo` | Exclude ruflo agents/skills/hooks (~40-60K fewer tokens) |
| `--local` | Install to `master/.claude/` instead of `~/.claude` |
| `--dry-run` | Preview only, no changes |
